# IS Section 3 Bot

A Telegram bot designed for managing assignments, exams, and announcements for IS Section 3. The bot provides features for both students and admins, including assignment submission, exam scheduling, and broadcasting messages.

## How It Works

The bot operates through a simple command-based interface on Telegram. Users interact with the bot by sending commands or using buttons provided in the chat. Here's a quick overview:

### For Students:

1. **Submit Assignments**: Students can upload their assignments directly to the bot. The bot organizes submissions by assignment type and group/individual.
2. **Exam Notifications**: Students receive timely notifications about upcoming exams and can view the schedule at any time.
3. **Interactive Features**: Engage with the bot through fun commands like "Buy Me Coffee."

### For Admins:
1. **Manage Exams**: Admins can add, delete, or reschedule exams with ease.
2. **Monitor Submissions**: View and download submitted assignments for review.
3. **Broadcast Updates**: Send important announcements to all users instantly.

The bot ensures a seamless experience for both students and admins, making academic management efficient and hassle-free.

## Technologies Used:

The bot is built using the following technologies and libraries:

- **Python**: The primary programming language for the bot's backend.
- **python-telegram-bot**: A library for interacting with the Telegram Bot API.
- **Heroku**: For deployment and hosting (optional).

## Features

### For Students:
- **Submit Group Assignment**: Upload group assignment files.
- **Submit Individual Assignment**: Upload individual assignment files.
- **View Exam Announcements**: See the list of scheduled exams.
- **Buy Me Coffee**: A fun feature to interact with the bot.

### For Admins:
- **Add Exam Date**: Schedule new exams with details like name, date, time, and content.
- **Delete Exam**: Remove scheduled exams.
- **Reschedule Exam**: Modify the date and time of an existing exam.
- **View Assignments**: Access submitted assignments.
- **Broadcast Messages**: Send announcements to all users.
